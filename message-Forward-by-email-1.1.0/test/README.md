## 🔍 测试详情

### 1. SMTP配置测试 (`SmtpConfigTest.kt`)

验证邮件服务配置的创建、验证和兼容性：

- **Gmail配置** - smtp.gmail.com:587 STARTTLS
- **QQ邮箱配置** - smtp.qq.com:465 SSL  
- **Mailpit测试** - 10.0.2.2:1025 无加密
- **无效配置检测** - 空字段、错误端口等
- **安全类型支持** - SSL/STARTTLS/NONE

### 2. 短信解析测试 (`SmsParserTest.kt`)

验证短信内容的解析、提取和隐私保护：

- **OTP验证码提取** - 4-8位数字边界检测
- **手机号掩码** - 138****5678 格式
- **长数字掩码** - 银行卡号、身份证号等
- **邮箱地址掩码** - a*c@domain.com 格式
- **复合信息处理** - 多种敏感信息同时掩码

### 3. 规则引擎测试 (`SmsFilterTest.kt`)

验证短信过滤和分类的智能处理：

- **白名单过滤** - 银行(95xxx)、运营商(10xxx)
- **黑名单过滤** - 广告、推广、退订等关键词
- **OTP消息识别** - 验证码类短信智能识别
- **消息格式化** - "验证码: 123456" 标准格式
- **自定义规则** - 用户自定义白/黑名单支持

## 🚀 端到端测试流程

### 1. 环境准备
```bash
# 启动Mailpit测试服务器
docker run --rm -p 8025:8025 -p 1025:1025 axllent/mailpit
```

### 2. 编译安装
```bash
# 编译APK (需要Android SDK)
./gradlew assembleDebug

# 安装到模拟器
adb install app/build/outputs/apk/debug/app-debug.apk
```

### 3. 配置测试
1. 打开应用 → 设置
2. 点击"Mailpit"预设按钮
3. 目标邮箱填写: `test@example.com`
4. 保存配置

### 4. 功能验证
```bash
# 发送测试短信
adb emu sms send +8613800138000 "【招商银行】验证码123456，5分钟内有效"

# 查看邮件
open http://localhost:8025
```
