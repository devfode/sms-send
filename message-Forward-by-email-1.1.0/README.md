# 短信转发云 SMS Email Relay

一个专为海外用户和多卡用户设计的Android短信邮件转发应用。

## 🌍 适用场景

### 海外用户
- **异国无法收到短信**：在海外时，国内手机卡经常无法正常接收短信
- **验证码获取困难**：银行、支付宝、微信等重要服务的验证码无法及时收到
- **重要通知遗漏**：工作、家庭、财务等重要短信可能因信号问题错过

### 多卡用户  
- **设备携带不便**：多张手机卡但不便携带多台设备
- **商务个人分离**：工作卡和个人卡需要同时监控但只想携带一台主设备
- **备用卡管理**：备用卡放在固定位置，需要远程获取短信内容

## ✨ 核心功能

### 🚀 全量短信转发
- **零过滤转发**：接收到的所有短信都会转发到邮箱，确保不遗漏任何重要信息
- **实时推送**：短信到达后立即通过邮件转发，延迟极低
- **多卡支持**：自动识别双卡手机，标记短信来源卡槽

### 🔒 隐私保护
- **智能脱敏**：自动识别并部分遮挡敏感信息（身份证、银行卡号等）
- **三级隐私等级**：
  - 低级：仅遮挡卡号和身份证
  - 中级：遮挡电话、卡号、身份证
  - 高级：遮挡所有敏感信息
- **加密存储**：配置和记录均采用AES256加密存储

### 📧 邮件配置
- **SMTP支持**：支持所有主流邮箱服务商（Gmail、Outlook、QQ邮箱等）
- **安全连接**：支持SSL/TLS加密传输
- **自定义发件人**：可自定义发件人名称，便于识别来源

### 📱 用户体验
- **Material Design 3**：现代化界面设计
- **自动权限管理**：智能引导用户开启必要权限
- **后台保活**：通过前台服务确保服务稳定运行
- **开机自启**：系统启动后自动恢复服务

### 📊 详细记录
- **转发历史**：完整记录所有短信处理状态
- **统计信息**：今日接收量、成功率等关键指标
- **状态追踪**：实时显示每条短信的处理状态（接收中、排队中、已发送、失败）

## 📋 系统要求

- Android 6.0 (API level 23) 或更高版本
- 通知访问权限
- 网络连接权限
- 前台服务权限

## 🛠️ 安装使用

### 下载安装
1. 从 [Releases](https://github.com/你的用户名/短信转发云/releases) 页面下载最新版APK
2. 允许安装未知来源应用
3. 安装并启动应用

### 配置步骤
1. **开启权限**：按提示开启通知监听权限和电池优化白名单
2. **配置SMTP**：
   - 邮箱服务器地址（如smtp.gmail.com）
   - 端口号（通常SSL为465，TLS为587）
   - 用户名和密码（建议使用应用专用密码）
   - 接收邮箱地址
3. **设置隐私等级**：根据需要选择合适的脱敏级别
4. **测试转发**：发送测试短信验证配置正确

## 🔧 常见邮箱配置

### Gmail
- 服务器：smtp.gmail.com
- 端口：465 (SSL) 或 587 (TLS)
- 需要开启两步验证并使用应用专用密码

### QQ邮箱
- 服务器：smtp.qq.com  
- 端口：465 (SSL) 或 587 (TLS)
- 需要开启SMTP服务并获取授权码

### Outlook
- 服务器：smtp.live.com
- 端口：587 (TLS)
- 使用Microsoft账户密码
